# Interactive Dashboard Project

Excel | Pivot Tables | Slicers | Interactive Dashboard | Sales Analysis

## Project Summary

This project shows how raw sales data was transformed into an interactive Excel dashboard for analysis and reporting.  
Starting with a raw data sheet, I created pivot tables to summarize the data and then built an interactive dashboard using slicers to filter results by **Year** and **Category**.

The final dashboard makes it easier to explore sales trends, compare sales performance across items, and present insights in a simple visual format.

---

## Project Objective

The objective of this project was to turn raw sales data into a structured and interactive reporting solution that helps users analyze performance more efficiently.
